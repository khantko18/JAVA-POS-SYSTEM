# 프로젝트 주요 기능 (Project Main Functions)

## Cafe POS System - Quick Reference

---

## 📋 주문 처리 (Order Processing)

| 주요기능          | 설명                                                                            | 비고                                  |
| ----------------- | ------------------------------------------------------------------------------- | ------------------------------------- |
| **상품 불러오기** | [제품 불러오기] 버튼을 눌러 구입 기능의 상품 목록을 상품 콤보 박스에 불러 온다. | 제고 현황에 등록된 상품 정보를 가져옴 |
| **상품**          | 구입 기능의 상품이 목록으로 보인다.                                             | 콤보 박스에서 구매할 상품 선택        |
| **수량**          | 구매 수량을 입력한다.                                                           | 숫자만 입력 가능                      |
| **추가**          | 구매하는 상품의 총 가격이 표시된다.                                             | 실시간 계산 및 업데이트               |
| **총가격**        | 구매 상품과 수량, 가격이 표시된다.                                              | 할인 적용 후 최종 금액                |
| **결제**          | [결제] 버튼을 눌러 지불할 금액을 입력 받는다.                                   | 현금/카드 선택 가능                   |
| **취소**          | [취소] 버튼을 눌러 선택한 상품이 모두 취소되어 화면이 초기화 된다.              | 확인 다이얼로그 표시 후 초기화        |

---

## ☕ 메뉴 관리 (Menu Management)

| 주요기능          | 설명                                             | 비고                                     |
| ----------------- | ------------------------------------------------ | ---------------------------------------- |
| **제품 불러오기** | 상품 정보가 테이블에 표시된다.                   | ID, 이름, 카테고리, 가격, 설명 자동 로드 |
| **제품 등록**     | [신규] 버튼을 누르면 제품 정보를 입력할 수 있다. | 자동 ID 생성 (M001, M002...)             |
| **제품 수정**     | 테이블에서 선택한 제품의 정보를 수정한다.        | 테이블 클릭 시 폼에 자동 로드            |
| **제품 삭제**     | [삭제] 버튼을 눌러 선택한 상품을 제거한다.       | 삭제 확인 메시지 표시                    |
| **입력 검증**     | 이름 필수, 가격 양수 검증                        | 오류 발생 시 경고 메시지                 |
| **카테고리 선택** | 커피, 음료, 디저트, 음식 중 선택                 | 드롭다운으로 선택                        |

## 📊 매출 통계 (Sales Statistics)

| 주요기능      | 설명                                | 비고                      |
| ------------- | ----------------------------------- | ------------------------- |
| **총 매출액** | 전체 기간 누적 매출액 표시          | 초록색 카드로 강조 표시   |
| **오늘 매출** | 당일 매출액 실시간 업데이트         | 파란색 카드로 표시        |
| **주문 건수** | 당일 처리된 총 주문 수              | 노란색 카드로 표시        |
| **거래 내역** | 시간, 주문ID, 금액, 결제방식 표시   | 최근 거래부터 시간순 정렬 |
| **인기 메뉴** | 판매량 기준 상위 메뉴 표시          | 내림차순으로 순위 표시    |
| **자동 갱신** | 탭 전환 시 자동으로 데이터 새로고침 | 실시간 데이터 반영        |

## 💳 결제 처리 (Payment Processing)

| 주요기능      | 설명                            | 비고                            |
| ------------- | ------------------------------- | ------------------------------- |
| **총 금액**   | 할인 적용된 최종 결제 금액 표시 | 큰 글씨로 명확하게 표시         |
| **결제 방식** | 현금 또는 카드 선택             | 라디오 버튼으로 선택            |
| **받은 금액** | 현금 결제 시 받은 금액 입력     | 현금 선택 시만 입력 가능        |
| **거스름돈**  | 거스름돈 자동 계산 및 표시      | 실시간 계산 (받은금액 - 총금액) |
| **결제 확인** | 결제 완료 시 영수증 정보 표시   | 주문ID, 금액, 거스름돈 포함     |
| **금액 검증** | 받은 금액이 총액 이상인지 확인  | 부족 시 에러 메시지             |

## 🌐 다국어/통화 지원 (Language & Currency)

| 주요기능          | 설명                           | 비고                     |
| ----------------- | ------------------------------ | ------------------------ |
| **언어 전환**     | English ↔ 한국어 즉시 전환     | 화면 상단 버튼 클릭      |
| **UI 번역**       | 모든 텍스트가 선택 언어로 변경 | 200개 이상 번역 키       |
| **통화 변환**     | $ (달러) ↔ ₩ (원) 자동 변환    | 환율 1:1200 적용         |
| **카테고리 번역** | Coffee↔커피, Beverage↔음료 등  | 표시만 번역, 저장은 영어 |
| **가격 형식**     | 언어별 통화 형식 자동 적용     | $3.50 ↔ ₩4,200           |

## 🏗️ MVC 아키텍처 (MVC Architecture)

| 구성 요소                 | 클래스                                                            | 역할                          |
| ------------------------- | ----------------------------------------------------------------- | ----------------------------- |
| **Model (모델)**          | MenuItem, Order, Payment, SalesData, MenuManager                  | 데이터 및 비즈니스 로직       |
| **View (뷰)**             | MainView, OrderView, MenuManagementView, SalesView, PaymentDialog | 사용자 인터페이스 (GUI)       |
| **Controller (컨트롤러)** | MenuController, OrderController, SalesController                  | 모델과 뷰를 연결, 이벤트 처리 |
| **Utility (유틸리티)**    | LanguageManager, LanguageResources                                | 다국어 지원 시스템            |

## 💪 주요 기술 특징 (Key Technical Features)

| 기술/패턴       | 설명                              | 장점                       |
| --------------- | --------------------------------- | -------------------------- |

| **실시간 계산** | 모든 금액 즉시 계산 및 표시       | 빠른 응답 속도             |

## 📌 시연 시나리오 (Demo Scenario)

### 1단계: 주문 처리

```
① 주문 탭 열기
② 카테고리 선택 (커피)
③ 메뉴 선택 및 수량 입력
④ 추가 버튼 클릭
⑤ 할인 적용 (선택사항)
⑥ 총액 확인
```

### 2단계: 결제

```
① "결제 진행" 버튼 클릭
② 결제 방식 선택 (현금/카드)
③ 받은 금액 입력 (현금 시)
④ 거스름돈 확인
⑤ "결제 확인" 클릭
⑥ 영수증 정보 확인
```

### 3단계: 관리 기능

```
① 메뉴 관리 탭에서 제품 추가/수정/삭제
② 매출 통계 탭에서 데이터 확인
③ 언어 전환 (English ↔ 한국어)
④ 통화 변환 확인 ($ ↔ ₩)
```
